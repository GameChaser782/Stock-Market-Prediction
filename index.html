What it does

Everything a financial advisor needs

Eight specialized agents working together — no API key required to start (uses Ollama by default).

⚖️

Bull vs Bear Debates

Two AI agents argue opposing cases on any stock with real data, rebuttals, and a moderator verdict with confidence score.

📊

Full Stock Analysis

Researcher + Analyst + Risk Assessor work together on fundamentals, technicals, and portfolio risk scoring.

📰

News Impact Analysis

Connects global macro events (tariffs, rate decisions, geopolitics) to specific stocks and sectors in your portfolio.

🤖

ML Price Predictions

XGBoost classifier trained on OHLCV + technical indicators predicts 5-day price direction. Train on any ticker.

💬

Streaming Chat

Ask anything about your portfolio. The supervisor agent routes to the right specialist and streams the response token by token.

🌐

Any LLM Provider

Ollama, Gemini, Claude, OpenAI — all treated equally. Pick on first run via setup wizard. No default, no lock-in.

Get started

Up and running in minutes

No financial data API key required — yfinance works out of the box. Add Finnhub for faster data.

1

Install

Requires Python 3.11+. Install the package and its dependencies.

pip install portfolioiq

For ML support: pip install "portfolioiq[ml]"

2

Run the setup wizard

Pick your LLM provider and enter API keys. Stored in .env — never committed.

portfolioiq setup

Providers: Ollama (local, free) · Gemini · Claude · OpenAI

3

Start the server

Starts FastAPI backend and serves the React frontend at localhost:8000.

portfolioiq serve
4

Run a stock debate (optional CLI)

Runs a 3-round bull vs bear debate directly in your terminal.

portfolioiq debate AAPL
5

Train ML models (optional)

Train a price direction classifier on any ticker, then predict.

portfolioiq train --ticker AAPL --days 365 portfolioiq predict AAPL
Architecture

Eight specialized agents

Supervisor routes queries. Each sub-agent has a YAML config and a markdown persona file in agents/.

supervisor

Routes user queries to the right specialist. Synthesizes multi-agent responses into a single coherent answer.

🔍 Researcher

Gathers raw financial data — prices, fundamentals, earnings, news — using the tool registry.

📈 Analyst

Technical and fundamental analysis. Calculates indicators, interprets patterns, and scores the stock.

bull Debater

Argues the bullish case with specific evidence: P/E ratios, revenue growth, momentum signals, positive news catalysts.

bear Debater

Argues the bearish case: debt levels, insider selling, competitive threats, macro headwinds.

mod Moderator

Judges the debate, scores argument quality, and outputs a structured verdict with confidence score (0–100).

📰 News Analyst

Maps global news events to stock and sector impact. Answers "how do tariffs affect my portfolio?"

⚠️ Risk Assessor

Portfolio-level risk: concentration, volatility, drawdown, correlation analysis.

API Reference

REST API

Full OpenAPI docs available at /docs when the server is running.

Method Endpoint Description
GET /api/health Server health + active provider/model
GET /api/search?q= Symbol autocomplete (no API key needed)
GET /api/quote/{ticker} Current price, change, currency
POST /api/stream SSE streaming chat (send message, get token stream)
GET /api/debate/{ticker} Run bull/bear debate. ?rounds=1-5
GET /api/analyze/{ticker} Full analysis: fundamentals + technicals + score
GET /api/news/{ticker} News impact analysis
GET /api/sessions List saved portfolio sessions
POST /api/sessions Create a new session
GET /api/sessions/{id} Load a session
PUT /api/sessions/{id} Save/update a session
DEL /api/sessions/{id} Delete a session

Contributing

The codebase is designed to be easy to extend. Each agent, tool, and graph is an isolated module.

Adding a new agent

Create agents/my_agent.yaml and my_agent_persona.md, implement the node function in portfolioiq/graphs/, and wire it into the supervisor.

Adding a new tool

Add a file in portfolioiq/tools/ with a @tool decorated function. Import it in the agent's YAML tool list. The registry picks it up automatically.

Adding a data provider

Extend portfolioiq/data/provider.py. The DataProvider class has a clean interface: get_quote(), get_fundamentals(), get_candles(), etc.
