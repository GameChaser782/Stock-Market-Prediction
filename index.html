Documentation

PortfolioIQ is a multi-agent financial advisor framework that allows you to analyze stocks, run bull/bear debates, and manage portfolios with ease. The main entry point is AgentKit, exported from agent_starter.

Repository

The source code is available on GitHub. You can contribute by opening issues or submitting pull requests.

GitHub Repository Download v0.1.0

Quickstart

Working agent in under a minute. Default flow: clone the repo, install in editable mode, start Ollama, then run the bundled basic example or the CLI.

git clone https://github.com/GameChaser782/agent-starter.git
cd agent-starter
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# .venv\Scripts\activate   # Windows

pip install -e .
cp .env.example .env
ollama serve
ollama pull qwen3.5
python examples/01_basic/run.py

What you get

Agent orchestration, persistent memory, built-in tools, CLI commands, and a FastAPI surface.

Installation

To use PortfolioIQ in your own project, we recommend using a virtual environment.

python -m venv .venv
source .venv/bin/activate
pip install portfolioiq

For ML features (predictions and training), install the optional dependencies:

pip install "portfolioiq[ml]"

Core Usage

PortfolioIQ is built around the AgentKit. You can initialize the supervisor agent and start interacting with it using the streaming API.

from portfolioiq import AgentKit

agent = AgentKit()
for token in agent.stream("Analyze @AAPL"):
    print(token, end="", flush=True)

Built-in Tools

Our agents have access to a variety of financial tools:

  • Researcher: Fetches historical prices and fundamentals.
  • News Search: Real-time news analysis using Tavily.
  • Indicators: Calculates RSI, MACD, and Bollinger Bands.
  • ML Predictor: XGBoost-based price direction prediction.

Examples

Check the examples/ directory in the repository for common use cases:

  • 01_basic/: Simple chat agent.
  • 02_debate/: Running a multi-round bull vs bear debate.
  • 03_portfolio/: Full portfolio risk assessment.

API Server

PortfolioIQ includes a built-in FastAPI server to power your web applications.

portfolioiq serve --port 8000

The following endpoints are available:

  • GET /api/health: Server status.
  • POST /api/stream: Streaming chat endpoint.
  • GET /api/debate/{ticker}: Run a stock debate.
  • GET /api/analyze/{ticker}: Full financial analysis.

Configuration

Configure your environment variables in a .env file. The setup wizard can help you with this:

source .venv/bin/activate
portfolioiq setup

Key configuration options:

  • MODEL_PROVIDER: ollama, openai, anthropic, or google.
  • MODEL_NAME: The specific LLM to use (e.g., qwen3.5, gpt-4o).
  • TAVILY_API_KEY: Required for live news search.
